What is the Document Object Model (DOM)?

The DOM(Document Object Module)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of objects (nodes), where each node corresponds to a part of the document, such as elements, attributes, or text content. Through the DOM, programs (usually JavaScript) can interact with, manipulate, and change the structure, style, and content of a webpage dynamically.

What is event delegation in the context of the DOM, and why is it useful?

The The Event Deligation

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of attaching individual event listeners to each child element, the parent listens for events that "bubble up" from the children, making the parent responsible for handling those events.
